Abstract (Introduction)

CineReserve Pro is a comprehensive cinema reservation system designed to streamline movie management and ticket booking processes for both administrators and customers. The system enables administrators to efficiently manage movies, configure showtimes, and organize theater rooms, including support for different seating classes such as Standard and Gold Class. It also incorporates real-time seat availability validation and dynamic updates to the theater seating map to ensure accurate booking information.

From the customer perspective, the system provides an intuitive interface for browsing available movies, selecting preferred showtimes, choosing seats, and receiving tickets upon successful booking. The system employs object-oriented design principles, including encapsulation, inheritance, aggregation, composition, and polymorphism, to ensure modularity and scalability. Notably, the ticketing mechanism supports multiple ticket types, such as adult and student tickets, with flexible pricing strategies.

Overall, CineReserve Pro enhances the efficiency, reliability, and user experience of cinema reservation systems by integrating core functionalities into a cohesive platform, making it suitable for modern digital ticketing environments.
